layoutWrap

Determines whether this layer should use wrapping auto-layout. Defaults to "NO_WRAP".

Supported on:

  • ComponentNode
  • ComponentSetNode
  • FrameNode
  • InferredAutoLayoutResult
  • InstanceNode
  • SlideNode

Signature

layoutWrap: 'NO_WRAP' | 'WRAP'

Remarks

This property can only be set on layers with layoutMode === "HORIZONTAL". Setting it on layers without this property will throw an Error.

This property must be set to "WRAP" in order for the counterAxisSpacing and counterAxisAlignContent properties to be applicable.
