id

The unique identifier of a node. For example, 1:3. The node id can be used with methods such as figma.getNodeByIdAsync, but plugins typically don't need to use this since you can usually just access a node directly.

Remarks

One possible use case for using the id is to have a serializable representation of a Figma node. To "deserialize" an id back into a node, pass it to figma.getNodeByIdAsync. This will return null if the node is no longer present in the document.

In the URLs for Figma files, node ids are hyphenated. However, for use with the API, node ids must use colons. For example, if a Figma file URL has the node id 1-3, you must convert it to 1:3.
