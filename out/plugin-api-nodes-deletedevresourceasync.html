deleteDevResourceAsync

Deletes a dev resource on a node. This will fail if the node does not have a dev resource with the same url.

Supported on:

  • BooleanOperationNode
  • CodeBlockNode
  • ComponentNode
  • ComponentSetNode
  • ConnectorNode
  • DocumentNode
  • EllipseNode
  • EmbedNode
  • FrameNode
  • GroupNode
  • HighlightNode
  • InstanceNode
  • InteractiveSlideElementNode
  • LineNode
  • LinkUnfurlNode
  • MediaNode
  • PageNode
  • PolygonNode
  • RectangleNode
  • SectionNode
  • ShapeWithTextNode
  • SliceNode
  • SlideGridNode
  • SlideNode
  • SlideRowNode
  • StampNode
  • StarNode
  • StickyNode
  • TableNode
  • TextNode
  • TextPathNode
  • TransformGroupNode
  • VectorNode
  • WashiTapeNode
  • WidgetNode

Signature

deleteDevResourceAsync(url: string): Promise<void>

Parameters

url

The url of the dev resource.
