# zoom

Zoom level. A value of 1.0 means 100% zoom, 0.5 means 50% zoom.

## Signature

### zoom: number

## Remarks

- zoom < 1: design is zoomed out
- zoom = 1: design is shown at exact size
- zoom > 1: design is zoomed in
