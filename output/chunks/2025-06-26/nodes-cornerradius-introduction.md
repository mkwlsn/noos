# cornerRadius

The number of pixels to round the corners of the object by.

 Supported on:

- BooleanOperationNode
- ComponentNode
- ComponentSetNode
- EllipseNode
- FrameNode
- HighlightNode
- InstanceNode
- PolygonNode
- RectangleNode
- SlideNode
- StarNode
- VectorNode
