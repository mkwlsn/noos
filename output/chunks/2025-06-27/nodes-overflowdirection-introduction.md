# overflowDirection

Determines whether a frame will scroll in presentation mode when the frame contains content that exceed the frame's bounds. Reflects the value shown in "Overflow Behavior" in the Prototype tab.

 Supported on:

- ComponentNode
- FrameNode
- InstanceNode
