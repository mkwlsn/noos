# cornerSmoothing

A value that lets you control how "smooth" the corners are. Ranges from 0 to 1.

 Supported on:

- BooleanOperationNode
- ComponentNode
- ComponentSetNode
- EllipseNode
- FrameNode
- HighlightNode
- InstanceNode
- PolygonNode
- RectangleNode
- SlideNode
- StarNode
- VectorNode
