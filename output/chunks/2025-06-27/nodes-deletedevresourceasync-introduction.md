# deleteDevResourceAsync

Deletes a dev resource on a node. This will fail if the node does not have a dev resource with the same url.

 Supported on:

- BooleanOperationNode
- CodeBlockNode
- ComponentNode
- ComponentSetNode
- ConnectorNode
- DocumentNode
- EllipseNode
- EmbedNode
