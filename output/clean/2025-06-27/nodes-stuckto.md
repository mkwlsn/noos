# stuckTo

If this node is stuck to another node this property returns that node or null.

 Supported on:

- HighlightNode
- StampNode
- WashiTapeNode
- WidgetNode

## Signature

### stuckTo: SceneNode | null

## Remarks

This property is only available in FigJam.
